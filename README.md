# chess

Chess puzzle solver

## Usage

```
$ lein run {FEN notation} {number of moves}
# i.e.
$ lein run "3k4/8/4K3/8/8/8/2R4P/8 w ---- - 0 1" 2
1. h3 Ke8 2. Rc8#
1. h4 Ke8 2. Rc8#

```
